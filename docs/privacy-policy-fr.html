Politique de Confidentialité - Mi Cesta Pro

Dernière mise à jour : 3 novembre 2025

1. Aperçu

Mi Cesta Pro est une application de listes de courses qui respecte votre vie privée. Cette politique explique comment nous traitons vos informations personnelles.

2. Informations que nous collectons

Informations que nous NE collectons PAS :

  • Nous ne collectons pas d'informations personnellement identifiables
  • Nous n'accédons pas à vos informations de contact
  • Nous ne suivons pas votre localisation
  • Nous ne collectons pas de données d'utilisation ou d'analyses

Informations locales :

  • Vos listes de courses sont stockées localement sur votre appareil
  • Les produits scannés sont sauvegardés dans la base de données locale
  • Les paramètres de l'application sont conservés sur votre appareil

3. Synchronisation iCloud

Mi Cesta Pro utilise iCloud d'Apple pour synchroniser vos listes entre vos appareils :

  • Les données sont synchronisées via votre compte iCloud personnel
  • Apple gère le chiffrement et la sécurité de ces données
  • Nous n'avons pas accès à vos informations iCloud
  • Vous pouvez désactiver la synchronisation à tout moment

4. Scanner de codes-barres

Lorsque vous scannez des produits :

  • L'application utilise la caméra pour lire les codes-barres
  • Elle interroge la base de données publique Open Food Facts pour les informations produit
  • Aucune image de la caméra n'est stockée ou partagée
  • Seul le code-barres est traité, pas les images personnelles

5. Reconnaissance vocale

Pour la fonction de dictée vocale :

  • Le traitement vocal est effectué localement sur votre appareil
  • Aucun audio n'est envoyé à des serveurs externes
  • Les commandes vocales ne sont pas stockées ou enregistrées

6. Partage d'informations

Nous ne partageons, vendons ou transférons pas vos informations personnelles à des tiers. L'application fonctionne entièrement hors ligne sauf pour :

  • Les requêtes optionnelles à Open Food Facts pour les informations produit
  • La synchronisation iCloud optionnelle (gérée par Apple)

7. Sécurité des données

  • Toutes les données sont stockées localement sur votre appareil
  • La synchronisation iCloud utilise le chiffrement standard d'Apple
  • Nous ne maintenons pas de serveurs avec vos informations personnelles

8. Confidentialité des enfants

Mi Cesta Pro est sûr pour tous les âges. Nous ne collectons pas d'informations personnelles des enfants ou de tout utilisateur.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique occasionnellement. Les changements seront notifiés via l'App Store et publiés sur cette page.
